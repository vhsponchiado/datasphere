# 📦 Base Dependencies

This template uses modern, high-performance technologies for the Node.js ecosystem.

## Core Technologies

- **[NestJS](https://nestjs.com/)**: Framework for building efficient, scalable server-side applications.
- **[Drizzle ORM](https://orm.drizzle.team/)**: Typescript-first, lightweight, and performant ORM.
- **[PostgreSQL](https://www.postgresql.org/)**: Robust relational database.
- **[Vitest](https://vitest.dev/)**: Extremely fast test framework, compatible with Vite.
- **[Zod](https://zod.dev/)**: Schema validation with a focus on TypeScript.
- **[Passport & JWT](http://www.passportjs.org/)**: Authentication and route protection.
- **[Swagger/Scalar](https://swagger.io/)**: Interactive API documentation.

## Useful Scripts

- `yarn start:dev`: Runs the application in development mode with watch.
- `yarn test`: Runs all unit tests.
- `yarn lint`: Checks and fixes code pattern issues.
- `yarn build`: Generates the production bundle.
