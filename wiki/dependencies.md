# 📦 Dependências Base

Este template utiliza tecnologias modernas e de alto desempenho para o ecossistema Node.js.

## Tecnologias Principais

- **[NestJS](https://nestjs.com/)**: Framework para construção de aplicações do lado do servidor eficientes e escaláveis.
- **[Drizzle ORM](https://orm.drizzle.team/)**: ORM Typescript-first, leve e performático.
- **[PostgreSQL](https://www.postgresql.org/)**: Banco de dados relacional robusto.
- **[Vitest](https://vitest.dev/)**: Framework de testes extremamente rápido, compatível com Vite.
- **[Zod](https://zod.dev/)**: Validação de esquemas com foco em TypeScript.
- **[Passport & JWT](http://www.passportjs.org/)**: Autenticação e proteção de rotas.
- **[Swagger/Scalar](https://swagger.io/)**: Documentação interativa da API.

## Scripts Úteis

- `yarn start:dev`: Roda a aplicação em modo de desenvolvimento com watch.
- `yarn test`: Roda todos os testes unitários.
- `yarn lint`: Verifica e corrige problemas de padrões de código.
- `yarn build`: Gera o bundle de produção.
